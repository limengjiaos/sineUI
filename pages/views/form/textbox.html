基本表单组件，支持input和textarea，并在原生控件基础上进行了功能扩展，提供组合型输入框。

代码示例

基本用法

通过给input标签添加class属性btsp-textbox，来生成最简单的input输入框。 


<!-- data-options定义属性 -->
<input class="btsp-textbox" data-options="width:'200px',placeholder:'请输入'">
或者
<!-- data-api单独定义属性 -->
<input class="btsp-textbox" data-width="200px" data-placeholder="请输入">
密码输入框

通过给input标签添加class属性btsp-passwordbox，来生成最简单的password输入框。 


<input class="btsp-passwordbox" data-width="200px">
文本域

通过给input输入框添加data-multiline="true"，或者data-options中添加multiline:'true'，来生成最简单的textarea文本域。 


<!-- data-options定义属性 -->
<input class="btsp-textbox" data-options="multiline:true" data-width="200px">
或者
<!-- data-api单独定义属性 -->
<input class="btsp-textbox" data-multiline="true" data-width="200px">

API

基本属性

属性名 说明 类型 默认值
multiline 是否为文本域 Boolean false
label 文本标题 String ''
id 文本框（域）id 属性 String ''
name 文本框（域）name 属性 String ''
labelWidth 文本框（域）标题宽度，可为具体 px，也可为栅格布局类：'col-md-2' String ''
inputWidth 文本框（域）宽度，可为具体 px，也可为栅格布局类：'col-md-2' String ''
labelAlign 文本框（域）标题居中方式 String 'right'
readonly 文本框（域）readonly 只读属性 Boolean false
disabled 文本框（域）disabled 禁用属性 Boolean false
value 文本框（域）value 属性 String ''
placeholder 文本框（域）placeholder 属性 String ''
helpText 文本框（域）底部帮助文本 String ''
rows 文本域 rows 属性 String ''
cols 文本域 cols 属性 String ''
valid 表单校验规则，默认为 false 不进行校验
