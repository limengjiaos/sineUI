开关选择器，用于在两种状态间切换。

代码示例

基础用法

状态切换时会触发change事件。 


  <input class="btsp-switchbox">
  <input class="btsp-switchbox" data-value=true>
自定义value

value默认为true和false。 通过 trueValue和falseValue来分别定义选中与非选中的value。 


  <input class="btsp-switchbox" data-truevalue="1" data-falsevalue="0">
  <input class="btsp-switchbox" data-truevalue="1" data-falsevalue="0" data-value="1">
禁用/只读

禁用开关。有只读和禁用两种模式，其中禁用会禁用表单元素。 


  <!-- 禁用 -->
  <input class="btsp-switchbox" data-disabled=true>
  <!-- 只读 -->
  <input class="btsp-switchbox" data-readonly=true>

API

基本属性

属性名 说明 类型 默认值
label 文本标题 String ''
id 开关id属性 String ''
name 开关name属性 String ''
labelWidth 标题宽度，可为具体px，也可为栅格布局类：如col-md-2 String ''
inputWidth 开关区域宽度，可为具体px，也可为栅格布局类：如col-md-2 String ''
labelAlign 开关标题居中方式 String 'right'
readonly 是否只读 Boolean false
disabled 是否禁用 Boolean false
value 开关value属性 String/Number/Boolean false
helpText 底部帮助文本 String ''
trueValue 选中时的值，当使用类似 1 和 0 来判断是否选中时会很有用 String/Number/Boolean true
falseValue 没有选中时的值，当使用类似 1 和 0 来判断是否选中时会很有用 String/Number/Boolean false
