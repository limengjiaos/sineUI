具有数据收集、校验和提交功能的表单，包含复选框、单选框、输入框、下拉选择框等元素。

我们为表单提供了以下两种排列方式：
  • 水平排列（form-inline、form-columns）：标签和表单控件水平排列；
  • 垂直排列（form-vertical）：标签和表单控件上下垂直排列；

代码示例

行内排列（form-inline）

水平排列的一种形式。 


<form class="form-inline form-group-sm">
    <input class="btsp-textbox" data-options="id:'a',inputWidth:'170px',name:'a'">
    <input class="btsp-selectbox" data-options="id:'c',inputWidth:'170px',name:'c',width:'150px',data:[{key:'请选择',value:''},{key:'选项1',value:'1'},{key:'选项2',value:'2'}]">
    <input class="btsp-checkbox" data-options="id:'b',inputWidth:'170px',name:'b',data:[{key:'选项1',value:'1'},{key:'选项2',value:'2'}]">
</form>

API

表单操作方法

$('#form').form('method-name', params)
方法名 说明 参数
load 根据表单元素 name 属性填充表格数据，支持深度填充，如 user.name 表单数据对象
clear 清空表单内容 -
data 获取表单数据，同时有参数 noValid，设为 true 则不会进行表单校验，否则会先进性表单校验，不通过则返回 false，通过返回表单数据 noValid
