主要用于展示大量结构化数据。

何时使用
  • 当有大量结构化的数据需要展现时；
  • 当需要对数据进行排序、搜索、分页、自定义操作等复杂行为时。

代码示例

基础用法

表格的最简单用法。默认自带斑马纹，可通过表格参数striped控制。默认自带分页，可通过表格参数pagination控制。 


<table id="table1"></table>
                
$('#table1').table({
    pagination: false,//取消分页
    columns: [{
        field: 'id',
        title: 'Item ID'
    }, {
        field: 'name',
        title: 'Item Name'
    }, {
        field: 'price',
        title: 'Item Price'
    }],
    data: [{
        id: 1,
        name: 'Item 1',
        price: '¥ 1'
    }, {
        id: 2,
        name: 'Item 2',
        price: '¥ 2'
    }, {
        id: 3,
        name: 'Item 3',
        price: '¥ 3'
    }]
});

表格formatter及事件绑定

columns参数formatter配合events使用实现单个或多个事件绑定。 


<table id="table6"></table>
                  
$('#table6').table({
    pagination: false,
    columns: [{
        checkbox: true
    }, {
        field: 'id',
        title: 'Item ID'
    }, {
        field: 'name',
        title: 'Item Name'
    }, {
        field: 'price',
        title: 'Item Price',
        events: {
            'click .myclass': function(e, value, row, index){
                e.stopPropagation();//防勾选
                $.alert(value);
            }
        },
        formatter: function(value, row, index, field){
            var color;
            switch(value){
                case '2': color = 'green';
                break;
                case '0': color = 'red';
                break;
                default: color = 'blue';
                break;
            }
            return '<span class="myclass" style="color:'+color+'">¥ '+value+'</span>';
        }
    }],
    data:[
        {
            "id": 0,
            "name": "Item 0",
            "price": '0'
        },
        {
            "id": 1,
            "name": "Item 2",
            "price": '1'
        },
        {
            "id": 2,
            "name": "Item 1",
            "price": '2'
        }
    ]
});

表格事件

    onEventName: function (arg1, arg2, ...) {
        // ...
    }
})
$('#table').on('event-name.bs.table', function (e, arg1, arg2, ...) {
    // ...
})
属性事件名 jQuery事件名 说明 参数
onAll all.bs.table 所有事件都会触发 name：事件名
args：事件方法参数
onClickRow click-row.bs.table 单击行事件 row：行数据

field：点击的单元格属性名
onDblClickRow dbl-click-row.bs.table 双击行事件 row：行数据
$element：行所在 tr
field：点击的单元格属性名
onCheck check.bs.table 勾选数据时触发 row：行数据
$element：行所在 tr
onUncheck uncheck.bs.table 取消勾选数据时触发 row：行数据
$element：行所在 tr
onLoadSuccess load-success.bs.table 远程获取数据成功后触发 data：表格数据
onPageChange page-change.bs.table 改变页码或修改每页显示数量时触发 number：pageNumber
size：pageSize
